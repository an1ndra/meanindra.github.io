Install MySQL on Ubuntu 20.04 LTS and Ubuntu 21.10

Anindra April 11, 2022

MySQL is one of the popular database management system and open-source, it can be install of almost any platform example Windows, Linux, Mac. It is fast and simple SQL based system.

Prerequisites

  • A system running Ubuntu 20.04 LTS or Ubuntu 21.10
  • SSH client (Optional)
  • Terminal
  • Internet connection
  • sudo privileges

Install MySQL on Ubuntu 20.04 and Ubuntu 21.10

If want to download MySQL which one is based on Ubuntu 20.04 LTS or Ubuntu 21.10 then can follow same process. If you have Ubuntu server on cloud server example Amazon AWS or DigitalOcean then connect that server through SSH client ( Termins), if server is not in cloud server simply open terminal (Ctrl+Alt+T)

  1. Update and Upgrade Ubuntu packages: To do this run

    sudo apt update
sudo apt upgrade

    to update packages and to upgrade packages. Maybe after running sudo apt upgrade you need to press Y to continue upgrade your system.

  2. Install MySQL: After update complete run sudo apt install mysql-server this will install MySQL server to your system. After some time later it may ask to continue install MySQL, press Y and enter to continue install. Run

    mysql --version

    to check whether MySQL installation is successful or not.

  3. Configure MySQL: By default MySQL server is less secure, so after installation done we need to make it secure. Run

    sudo mysql_secure_installation

    this commands will take a series of option to make changes MySQL installationâ€™s security options.
