Install Telegram on Ubuntu, Pop-OS, Manjaro and other Linux distro

Anindra October 2, 2021 Install

Unlike WhatsApp, Telegram have desktop client for multiple devices. Telegram desktop clients are open-source and free to use, available on GitHub.

Features:

  1. Telegram available for 32bit and 64bit computer.
  2. Available in Linux, Windows, macOS, iPadOS, iPhone, Android.
  3. Telegram a combination of 256-bit symmetric AES encryption, 2048-bit RSA encryption, Diffie–Hellman secure key exchange. That make Telegram privacy focus application.
  4. Can share large file in chat.
  5. Telegram in free forever with no ads, no subscription charge.
  6. Telegram provide API that helps developer to make custom desktop.
  7. In their recent update users can change chat Themes, use many Interactive emojis, can read receipts in small groups also can record live streams and video chat.

Install for Ubuntu:

Ubuntu 20.04, Ubuntu 18.04, Ubuntu 21.o4 comes with pre-installed package manager called snap, so it is easy to install Telegram in Ubuntu. To install Telegram, run this following command

sudo snap install telegram-dekstop

This command will install the latest and stable version snap Telegram. For more info, visit this link: https://snapcraft.io/telegram-desktop

Or can download telegram via apt package manager. This is the easy way to download Telegram in ubuntu, but it may or  may not install the latest release of telegram.

sudo apt install telegram-dekstop

Install for Pop OS, Linux mint, Ubuntu and other Ubuntu based distro:

Go to official telegram website https://desktop.telegram.org/ and click on Get Telegram for Linux, this will download a tar.xz . Now extract this tar file

tar -xf tsetup-x64.3.1.1.exe

Another way

wget -O telegram.tar.xz https://updates.tdesktop.com/tlinux/tsetup.3.1.1.tar.xz
tar -xf telegram.tar.xz

Then go to telegram folder and Telegram folder and give read, write and execute permission to Telegram application

Install Telegram in Manjaro (any Arch based distro) and any Linux distro:

Flat-hub package manager available in most of the popular Linux distro like Ubuntu, Manjaro, Linux Mint Manjaro etc. So it is easy to install telegram , not Only telegram also other application. If you think flathub is not available in your Linux desktop then follow this article to install https://flatpak.org/setup/

To install telegram run this following command to your terminal

flatpak install flathub org.telegram.desktop
flatpak run org.telegram.desktop
